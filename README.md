This repo contains my "Wolfe" agent for the VGC AI Competition 2025.
It is based on the famous [AlphaZero](https://arxiv.org/pdf/1712.01815) algorithm and [Open-Loop MCTS](http://www.diego-perez.net/papers/OpenLoopGVG.pdf).

This agent won second place in the Championship Track[^1]. 

[^1]: [Website and results for the 2025 competition](https://gitlab.com/DracoStriker/pokemon-vgc-engine/-/wikis/VGC-AI-Competition/2025)
